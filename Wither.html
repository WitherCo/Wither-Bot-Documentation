Moderation Commands

Commands to help manage your server:

  • !ban [user] [reason] - Bans a user from the server.
  • !kick [user] [reason] - Kicks a user from the server.
  • !mute [user] [duration] - Mutes a user for a specified duration.
  • !unmute [user] - Unmutes a user.
  • !warn [user] [reason] - Issues a warning to a user.
  • !clear [number] - Deletes the specified number of messages from a channel.
  • !lock [channel] - Locks a channel, preventing users from sending messages.
  • !unlock [channel] - Unlocks a channel, allowing users to send messages.
  • !slowmode [seconds] - Sets the slowmode duration for a channel.

Ticket Commands

Commands to manage support tickets:

  • !ticket create [issue] - Creates a new ticket with the specified issue.
  • !ticket close [ticket_id] - Closes the specified ticket.
  • !ticket list - Lists all open tickets.
  • !ticket info [ticket_id] - Provides information about the specified ticket.
  • !ticket assign [ticket_id] [user] - Assigns a user to the specified ticket.
  • !ticket unassign [ticket_id] - Unassigns the user from the specified ticket.
  • !ticket reopen [ticket_id] - Reopens a closed ticket.

Fun Commands

Commands to add some fun to your server:

  • !joke - Tells a random joke.
  • !meme - Displays a random meme.
  • !roll [number] - Rolls a dice with the specified number of sides.
  • !8ball [question] - Answers a yes/no question with a random response.
  • !cat - Shows a random cat picture.
  • !dog - Shows a random dog picture.
  • !quote - Provides a random motivational quote.
  • !trivia - Starts a trivia game with random questions.

Utility Commands

Commands to provide useful utilities:

  • !ping - Checks the bot's latency.
  • !userinfo [user] - Displays information about the specified user.
  • !serverinfo - Displays information about the server.
  • !avatar [user] - Shows the avatar of the specified user.
  • !poll [question] - Creates a poll with the specified question.

Music Commands

Commands to manage music playback:

  • !play [song] - Plays the specified song.
  • !pause - Pauses the current song.
  • !resume - Resumes the paused song.
  • !skip - Skips the current song.
  • !queue - Displays the current song queue.
  • !np - Shows the currently playing song.
  • !stop - Stops the music and clears the queue.
